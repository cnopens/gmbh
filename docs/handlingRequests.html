Handling Requests

Handling data requests in gmbh is fairly straightforward. All request contain a Payload object to retrieve data from and either the same or a different Payload object is used to return data to the caller.

In Go

func handleData(req gmbh.Request, resp *gmbh.Responder) {

    // Get the data out of the request
    data := req.GetPayload()
    value := data.Get("<dataName>")             // Returns an interface{}
    value := data.GetAsString("<dataName2>")    // Returns data as a string

    // Create a response 
    payload := gmbh.NewPayload()
    payload.Append("<dataName>", <data>)
    
    // send the response
    resp.SetPayload(payload)
}
