Running gmbh Projects

Projects can be controlled via the command line.

Getting Started

From the directory that the gmbh config file is gmbh --run --config=<configFile>.toml

Logs

This will redirect all output to the logs which will be located in the ./gmbh/logs/ directory.

Verbose Mode

To redirect gmbhCore’s output to standard out, include the --verbose flag.


Shutting Down

To shutdown gmbh projects, signal the process using ctrl+c in the terminal and all associated processes will be shutdown. If gmbh-micro has been started in daemon mode, run gmbh --info --q for shutdown.


Advanced

Daemon Mode

To run gmbh in daemon mode specify the --daemon flag.

No Log Mode

To not log anything except what is directed to Standard out specify the --nolog flag.
