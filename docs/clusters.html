Managing Docker-Compose Clusters

gmbh-micro can genrate bare bones docker-compose clusters, including all Dockerfiles needed to build the appropriate images. These Dockerfiles can then be edited to suit the needs of the project.

To build the deployment scaffold, run
gmbh --build-deploy --config=configFile.toml --verbose

This will generate all of the neccesary files in the gmbh-deploy subdirectory including a build script that can be used to build all of the images specified in the docker-compose file.
