Using the Command Line Tools

gmbh-micro projects can be controlled from the command line. To use any of the command line functionality, first specify --info before any of the following flags.

command info
--info Lists info about all connected services and their Remote process managers.
--list-one=<remoteID>-<serviceID> Lists information about the service with the specified ID.
--restart Restarts all attached services.
--restart-one=<remoteID>-<serviceID> Restarts the service with the specified ID.
