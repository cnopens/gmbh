Getting Started Using Node

The easiest way to install the Node client is npm i gmbh

1) Instantiate and Configure the client

gmbh requires several parameters to be passed into the constructor for the client.
Required Parameters

client = new gmbh.gmbh();
client.opts.service.name = "";

2) Registering Routes

client.Route("RouteName", routeHandler)

Handler functions have the following type signature and return a payload object.
async function grantAuth(sender: string, request: any)

3) Starting the client

client.Start().then(()=>{
    // Returns promise after client started.
});
